## Запуск проекта
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/username/repository-name.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd repository-name/lab1
   ```
3. Запустите программу:
   ```bash
   python src/main.py
   ```

4. Запуск тестов:
   ```bash
   python src/main.py
   ```
